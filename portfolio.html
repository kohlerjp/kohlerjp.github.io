Æther

Æther is an iOS 8 application. It is a photo sharing application similar to SnapChat, where photos are shared randomly amongst friends. When a user sends an image out into the "Æther", they immediately receive a picture a friend has posted. I came up with the idea for Æther when trying to consider an application in which a user would need to produce content in order to consume content. Aether has a back-end written in Ruby on Rails, and a front-end written in Swift. Add me as a friend under KohlerExpress.

Scruffy Sounds

Music has always been a large part of my life. I created a music-based blog with Ruby on Rails to catalog and share my favorite songs. The blog allows users to comment on and 'bump' songs. I used Twitter bootstrap for the front end work. The blog is located at scruffysounds.com.

Join in

As part of the Entrepreneurs Club InnoWeekend, I worked together with a team to create an idea, a business plan, and a prototype, all in the span of 48 hours. We came up with the idea of join in, software that would gamify the college experience. Students would be awarded points for attending school events, which they could redeem for school merchandise. I was tasked with creating the prototype for the team. I built a website that allows users to create an account/profile, register for upcoming events, and see which events their friends are talking about/attending. The website is located here. Make a fake account and take a look around!

It's Five Somewhere

"It's Five Somewhere" is a web app I made using Ruby on Rails. The page displays a location where it is currently 5:00 PM, along with a popular beverage from that part of the world. The website can be seen at itsfivesomewhere.co.

Whisperer

For HackBeanpot 2014, a Boston based hack-a-thon, I worked together with a team to create a humble social network, one where users were not allowed to post about themselves. Users would log in through facebook, and choose a friend to talk about. After adding the functionality for posts to be anonymous, we decided to call it whisperer. We used the facebook open graph api, as well as a natural language processor, that would tint a post green if it was positive, or red if it was negative. The github repository can be seen here.
